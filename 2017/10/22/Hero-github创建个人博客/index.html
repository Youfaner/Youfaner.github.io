Hero+github.io搭建个人博客

发表于 2017-10-22 | 分类于 个人博客搭建

本文详细介绍Hero+github搭建个人博客的方法和注意事项。

环境配置准备

下载安装Node.js和Git，选择安装系统对应的版本。安装完毕后，在cmd（命令行）中输入
node –version
git –version

若像上图一样显示出Node.js和Git的版本，则安装正确！

申请GitHub账号

在GitHub官网注册一个账号，并且新建一个仓库，仓库名一定要是yourname.github.io，其中yourname是你的用户名。同时，在命令行中执行
ssh-keygen -t rsa -C “youremail”（youremail为你注册的邮箱）一直按“Enter”生成密钥。将用户目录下的.ssh子目录中的id_rsa PUB文件内容
复制到GitHub账号“seting”下的“SSH GPL”,添加ssh密钥。还要执行命令ssh -T git@github.com验证是否添加密钥成功。最后执行命令
git config –global user.name “yourname”
git config –global user.email “youremail” (yourname youremail 为github账号名字和邮箱)配置git用户和邮箱。

下载安装Hero并配置

由于npm容易被墙，推荐使用cnpm。在cmd执行以下命令下载安装
$ npm install -g cnpm –registry=https://registry.npm.taobao.org
完成后即可利用cnpm下载安装Hero，首先新建一个文件夹作为博客根目录，比如myblog
cd命令切换到myblog目录中执行命令
cnpm install -g hexo-cli
cnpm install hexo –save
cnpm install hexo-deployer-git –save

hexo init
cnpm install

执行完毕后myblog目录下就有_config.yml配置文件，在该文件最后有deploy需要我们手动配置，配置如下
deploy:
type: git
repo: git@github.com:Youfaner/Youfaner.github.io.git(Youfaner改为你github账号的名字)
branch: master

Hero新建文章并发布

hero clean 清空缓存
hero new “title” 新建文章
hero d -g 或者 hero g -d 即可生成博客并发布
详细操作参考Hero官网
